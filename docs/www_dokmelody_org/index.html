Development status

Up to date, Dokmelody is in design and development phase.

Dok

Dok is programming language based on stepwise refinements of source code.

Doknil

Doknil is a simple knowledge-base language, for linking and navigating in all the parts of a project.

Tuttology

Tuttology is a desktop IDE for viewing and editing information stored in the Dokmelody servers.

Dokmelody

Dokmelody system is implemened in Dok and Doknil. Tuttology is its default IDE.

Federation

In Dokmelody every user, group or community can decide what to share, branch or fork.
