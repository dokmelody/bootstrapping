# About

DokMelody is an environment for analyzing and transforming knowledge. It aims to be:

* an adaptable project and knowledge management tool
* an IDE for programming in various programming languages
* a distributed network of shared documents
* a collaborating community of users and content creators
* a channel for funding content creators

It will be implemented in Dok, that is a new programming language for analyzing and transforming hierarchical (tree-like) data, and in Knil that is another new language for managing relational and graph-like data. 

Dok and DokMelody will be bootstrapped using the [Racket] programming language. 

[Racket]: http://racket-lang.org/
